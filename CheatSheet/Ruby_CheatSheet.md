## Ruby
~~~
# 基本的なRubyコマンド
バージョン確認                 : ruby -v
Rubyスクリプト実行             : ruby スクリプト名.rb
インタラクティブシェル(IRB) 起動: irb
Gem インストール               : gem install パッケージ名
Gem アンインストール           : gem uninstall パッケージ名
インストール済みGemの一覧表示   : gem list
Bundler インストール           : gem install bundler
Gemfile から Gem インストール  : bundle install
Rakeタスクの実行               : rake タスク名

~~~
